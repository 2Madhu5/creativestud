Animation Steps — Kling AI
STEP 1
Kling AI లో Login చేయండి
klingai.com → Sign up → "Image to Video" option select చేయండి
STEP 2
Image Upload చేయండి
Bing నుండి download చేసిన image upload చేయండి → ఒక్కొక్క image separately animate చేయండి
STEP 3
Animation Prompt ఇవ్వండి
ప్రతి image కి ఈ prompts వాడండి:

Battlefield image: "slow camera pan across the battlefield, dust particles floating, dramatic atmosphere"

Vyasa meditating: "gentle breeze moving sage's hair and robe, soft light flickering, peaceful"

Ganesha appearing: "slow zoom in on Ganesha, golden light pulsing gently, majestic and divine"

Writing scene: "slow zoom out revealing both characters, gentle lamp flame flickering, peaceful night"

Mountain final shot: "slow aerial pull back revealing the vast mountain range, stars twinkling"
STEP 4
Settings Select చేయండి
Duration: 5 seconds · Mode: Standard · Camera: Slow zoom or Pan
STEP 5
Download చేయండి
Generate అయిన video clip download చేయండి → "EP01_scene1.mp4" గా save చేయండి. ప్రతి image కి repeat చేయండి.
✅ 10 images × 5 seconds each = 50 seconds of animation. CapCut లో loop చేసి longer duration చేయవచ్చు!
🎞️
Combine Everything in CapCut
Site: capcut.com → Download PC version → New Project → 1920x1080
1
Animated Clips Import చేయండి
Kling AI నుండి download చేసిన అన్ని .mp4 clips import చేయండి → timeline లో scene order లో arrange చేయండి
2
Voice Import చేయండి
ttsmp3.com నుండి download చేసిన EP01_voice.mp3 import చేయండి → audio track లో add చేయండి
3
Clip Duration Adjust చేయండి
Voice audio తో clips sync అయ్యేలా చేయండి. Voice కి తగినట్టు clip duration extend లేదా trim చేయండి. CapCut లో clip right click → Loop చేయవచ్చు!
4
Background Music Add చేయండి
YouTube Audio Library → "Epic Indian" search → free track download → import → volume 20% గా set చేయండి
5
Telugu Subtitles Add చేయండి
CapCut → Text → Auto Captions → Telugu select → Generate. Font size 60, Bold, White color, Black outline
6
Transitions Add చేయండి
ప్రతి clip మధ్య Fade transition add చేయండి → duration 0.5 seconds. Dramatic scenes కి "Blur" transition వాడండి
7
Thumbnail తయారు చేయండి
canva.com → YouTube Thumbnail (1280x720) → Best Ganesha image background → మహాభారతం Episode 1 Telugu text in gold → Download PNG
8
Export చేయండి
CapCut → Export → Resolution: 1080p → Quality: High → Export! File size ~200-500MB
